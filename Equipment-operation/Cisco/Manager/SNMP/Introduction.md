# Simple Network Management Protocol 簡單網路管理協定 #

## 簡介 ##

    SNMP是幾乎所有企業與校園中都會啟用的協議，用於管理連上網路的裝置，使管理員能夠即時監控設備異常並迅速處理

## SNMP基本元件 ##

    SNMP管理的網路由以下三個元件組成
    1.Network-Management Systems(NMS)網路管理系統
    2.Managed Device管理裝置
    3.agent代理
